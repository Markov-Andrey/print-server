Print Server API

Header:
X-Token (str) — токен авторизации в заголовке запросов (обязательный). Пример: X-Token: SKYv8dG34g=

POST /print-svg Печать набора svg по заданным параметрам

Parameters:
printer (str) — имя принтера (обязательный)
width (int) — ширина в миллиметрах одной страницы/этикетки (обязательный)
height (int) — высота в миллиметрах (обязательный)
data (List[str]) — список SVG в base64 (обязательный)
grid (int) — количество ячеек по горизонтали (по умолчанию 1)
gap (int) — отступ между ячейками в мм (по умолчанию 0)
padding_x (int) — отступы слева/справа от границ страницы, мм (по умолчанию 0)
padding_y (int) — отступы сверху/снизу от границ страницы, мм (по умолчанию 0)

POST /print-doc Печать word файла (doc/docx)

Parameters:
printer (str) — имя принтера (обязательный)
filename (str) — имя файла (обязательный)
data (str) — содержимое файла в base64 (обязательный)

POST /print-file Печать файла

Parameters:
format (str) — расширение файла (обязательный), допустимые: JPG, JPEG, PNG, BMP, GIF, TIF, TIFF, WEBP, HEIC, AVIF, PDF, SVG, ICO, PSD, DJVU, TXT, EPS, PS, AI
printer (str) — имя принтера (обязательный)
filename (str) — имя файла (обязательный)
data (str) — содержимое файла в base64 (обязательный)
